2017/06/01 - Design a LINE Bot : Weather Bot!¶

My Weather Bot¶

Here is the link about my Weather Bot on Github.

Introduction about the design¶

I follow this tutorial [1] to build a LINE Bot on Heroku [2] - a free platform to run my node.js application with TLS certification. And I get the 天氣小幫手 message from 交通部中央氣象局開放資料平台 [3] , so that I can get latest information about weather today!

Reference¶

[1]LINE BOT 實戰 ( 原理篇 ) : http://www.oxxostudio.tw/articles/201701/line-bot.html
[2]Heroku : https://www.heroku.com/
[3]交通部中央氣象局開放資料平台 : http://opendata.cwb.gov.tw/
