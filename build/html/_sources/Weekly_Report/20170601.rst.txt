Introduction about the design
-------------------------------

I follow this tutorial [#]_ to build a LINE Bot on Heroku [#]_ - a free platform to run my node.js application with TLS certification. And I get the 天氣小幫手 message from 交通部中央氣象局開放資料平台 [#]_ , so that I can get latest information about weather today!
